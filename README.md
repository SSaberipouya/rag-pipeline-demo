# Mini RAG Project (Retrieval-Augmented Generation)

This project demonstrates a simple end-to-end RAG pipeline using:
- Sentence Transformers for embeddings
- FAISS for vector search
- OpenAI (optional) or mock generation

## Architecture

User Query → Embedding → Vector Search → Retrieved Context → LLM → Answer

## Features

- Semantic search using embeddings
- Context-grounded generation
- Simple and modular pipeline

## Sample Output

Retrieved context:
Hallucination is a key challenge in LLM systems.

Answer:
Hallucination refers to incorrect but confident outputs generated by the model.
